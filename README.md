# Guida Social Features

## Overview

The Guida Social Features repository encompasses the social aspects of the Guida app, including sign-up/sign-in processes, chatrooms, posting capabilities, and friend suggestions. This component aims to foster a community within the app, allowing users to connect, share experiences, and explore their surroundings together.

## Features

- **Authentication:** Secure sign-up and sign-in functionalities.
- **Chatrooms:** Real-time messaging and themed discussion rooms.
- **Posting:** Share experiences, photos, and insights about locations.
- **Friend Suggestions:** AI-powered recommendations to connect users with similar interests.
